# m-sport-resource

## 项目简介

`m-sport-resource` 是一个用于归档与发布多个独立 HTML 页面的资源仓库。本仓库不针对任何特定域名或网站，仅作为 HTML 文档的存储与分发点。

主要用途包括但不限于：保存静态页面快照、存放前端演示页面、提供离线可浏览的 HTML 资源集合。

## 目录说明

```
m-sport-resource/
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/` — 每个子目录或文件对应一个独立页面，文件名或目录名建议使用简短描述性名称。
- `assets/` — 存放页面所需的公共资源文件，可按类型或页面进一步组织。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开运行。
- 页面之间无依赖关系，可单独浏览或部署。
- 归档内容可能包含试验性功能、历史版本或临时演示页面，不保证长期可用或兼容性。
- 页面内容来源遵循原始版权说明，本仓库仅作存档用途。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面。
- 新增页面请放置在 `pages/` 目录下，并确保所有资源引用路径正确（建议使用相对路径）。
- 若页面依赖外部资源（如 CDN 库），请尽量使用稳定版本并注明来源。
- 不接受的贡献：包含恶意代码、侵犯版权内容、指向特定商业或政治宣传的页面。
- 本仓库由维护者不定期更新，不承诺固定的发布周期。
